# TikTok Live Socket

A socket application using nodejs to connect TikTok live streaming such as comments and gifts in realtime

## Installation

Tested on : 
- node `16.16.0`
- npm `8.11.10`

Run
```bash
npm install
```
or (to install based on `package-lock.json`) 
```bash
npm ci
```
